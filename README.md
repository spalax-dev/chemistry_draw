# Chemistry Draw

![./src-tauri/icons/icon.ico](Chemistry Draw)


[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

**Chemistry Draw** is a desktop wrapper of the chemical structure editor [Ketcher](https://github.com/epam/ketcher) (© EPAM Systems, Apache License 2.0) for Linux.

> Not an official EPAM product. Community-maintained packaging without affiliation or official support.

## What this wrapper offers

- Ketcher distributed as a standalone desktop application
- Integration with the system application menu
- Launchable from terminal or graphical interface
- Icon and metadata compatible with the Linux desktop
